## Installing
Using Npm:
```bash
$ npm install @ciceksepeti/cui-hooks
```
Using Yarn:
```bash
$ yarn add @ciceksepeti/cui-hooks
```

## Example

```jsx
import { useCombinedRefs } from '@ciceksepeti/cui-hooks';
```
