## Content

<InfoContainer>
  The Tabs component consists of clickable tabs, that are aligned side by side.
  Tabs make it easy to explore and switch between different views. Tabs organize
  and allow navigation between groups of content that are related and at the
  same level of hierarchy.

const Example = () => {
  return (
    <Tabs>
      <TabList>

        <Tab disabled>Tab2</Tab>

};

### Controlled

<AdmonitionContainer type='caution'>
  In order to use Tabs component as controlled, user have to give{' '}
  <span style={{ fontWeight: '600' }}>index</span> and{' '}
  <span style={{ fontWeight: '600' }}>onChange</span> properties at the same
  time.
  <br/>
  <br/>
  However, be sure that you have removed <span style={{ fontWeight: '600' }}>defaultIndex</span> prop while you are using controlled component.

</AdmonitionContainer>

```jsx
import React, { useState } from 'react';

import {
  Tabs,
  TabList,
  Tab,
  TabPanelList,
  TabPanel,
} from '@ciceksepeti/cui-tabs';
import '@ciceksepeti/cui-tabs/styles.css';

const Example = () => {
  const [key, setKey] = useState(0);
  return (
    <Tabs index={key} onChange={(k) => setKey(k)}>
      <TabList>
        <Tab>Tab1</Tab>
        <Tab disabled>Tab2</Tab>
        <Tab disabled>Tab3</Tab>
        <Tab>Tab4</Tab>
        <Tab disabled>Tab5</Tab>
      </TabList>
      <TabPanelList>
        <TabPanel>Panel1</TabPanel>
        <TabPanel>Panel2</TabPanel>
        <TabPanel>Panel3</TabPanel>
        <TabPanel>Panel4</TabPanel>
        <TabPanel>Panel5</TabPanel>
      </TabPanelList>
    </Tabs>
  );
};
```

### Render Props

> Example of a default behavior of Tabs component with render props pattern. Renders as **'div'** by default.

```jsx
import {
  Tabs,
  TabList,
  Tab,
  TabPanelList,
  TabPanel,
} from '@ciceksepeti/cui-tabs';
import '@ciceksepeti/cui-tabs/styles.css';

const Example = () => {
  return (
    <Tabs>
      {({ selectedTabIndex, focusedTabIndex }) => {
        return (
          <>
            <TabList>
              <Tab>Tab1</Tab>
              <Tab disabled>
                {focusedTabIndex === 0 ? 'focus on left' : 'focus on right'}
              </Tab>
              <Tab>{selectedTabIndex === 2 ? 'selected' : 'not selected'}</Tab>
            </TabList>
            <TabPanelList>
              <TabPanel>Panel1</TabPanel>
              <TabPanel>Panel2</TabPanel>
              <TabPanel>Panel3</TabPanel>
            </TabPanelList>
          </>
        );
      }}
    </Tabs>
  );
};
```
